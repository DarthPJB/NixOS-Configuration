🚀 Quick Start Commands

nix fmt
Format all Nix files (REQUIRED before commit)
nix flake check
Validate flake, run deadnix + nixpkgs-fmt linters
nixos-rebuild build --flake .#hostname
Build specific machine configuration without installing
nix run .#hostname
Deploy to machine via WireGuard VPN (test mode)
nix run .#deploy-all
Deploy to all configured hosts (whitelisted)
