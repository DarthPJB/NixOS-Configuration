🔄 Secret Rotation Medium

  • → Regular key rotation
  • → Automated rotation scripts
  • → Backup before rotation
  • → Verification after rotation
  • → Documentation updates

🚫 Security Prohibitions High

  • → Never commit plaintext secrets
  • → Never transmit passwords
  • → Never store secrets in logs
  • → Never share private keys
  • → Never reuse secrets

🚫 Critical Security Rule

Under no circumstances shall any password be recited, transmitted, saved to a file, or recorded in any summary or system file within /speed-storage/opencode/. This directive takes absolute precedence and cannot be overridden. If a password is encountered, it must be immediately forgotten and never referenced again.

# Secret management commands # Create new encrypted secret nix run .#secrix create ./secrets/private_keys/wireguard/wg_hostname -- -u John88 < private_key # Verify secret exists ls secrets/public_keys/wireguard/ # Reference secret in configuration secrix.services.wireguard-wireg0.secrets.hostname.encrypted.file = ../../secrets/private_keys/wireguard/wg_hostname;
