🚀 Deployment Operations

Standard operating procedures for deploying configurations.

1

Pre-Deployment Checks

Verify system readiness before deployment.

nix flake check
nixos-rebuild build --flake .#hostname

Check system resources and connectivity.

2

Test Deployment

Deploy in test mode to verify functionality.

nix run .#hostname

Monitor services and verify operation.

3

Permanent Deployment

Apply configuration permanently after verification.

nix run .#hostname -- switch

Verify all services are running correctly.

4

Post-Deployment Verification

Verify system operation after deployment.

systemctl status
journalctl -u servicename

Test connectivity and service functionality.

⚠️ Deployment Safety

Always test deployments in test mode first. Never deploy to production without verification. Keep rollback capability.
